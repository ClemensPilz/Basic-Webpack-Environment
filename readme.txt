Smallscale dev-environment to demonstrate basic webpack-capabilities

Requires: node.js, NPM

Setup:
• Navigate to your project folder in terminal/cmd
• Download and install dev-dependencies: npm install
• Start webpack and webpack-dev-server: npm run dev
• Open localhost:8080 in your webbrowser

Webpack will watch files for changes and update the output dynamically. To stop batch-process, press ctrl + c on Windows.

Resources: https://webpack.js.org/guides/installation/
